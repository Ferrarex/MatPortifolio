Conjuntos:

Gerais:

Conjuntos podem ser utilizados para expressar das coisas mais diversas, como intervalo entre números, naipes de carta, nomes...

Para representar um conjunto, podemos usar uma propriedade que o encaixará em algum sistema, ou escreve-lo por completo (quando for o caso e o conjunto não ser infinito).

Exemplo de representações:

A = {x | x é uma letra maiúscula do alfabeto Grego} ou
A = {A,B,Γ,Δ,E,Ζ,H,Θ,I,K,Λ,M,N,Ξ,O,Π,P,Σ,Τ,Y,Φ,X,Ψ,Ω} ou
A = {A,B,Γ,...} (caso o conjunto seja infinito, não é o caso, mas seria assim).

Nessa situação, o conjunto "A" é todas as letras maiúsculas do alfabeto Grego


Aqui são os símbolos utilizados e o que eles representam:

Separações:

Conjunto vazio: ∅ ou {}

Quando um conjunto não possui nenhum elemento dentro dele.


Pertence: ∈

Quando algo pertence a um conjunto, Ex: {A,B,Γ,Δ,E,Ζ,H,Θ} ∈ A


Não pertence: ∉

Quando algo não pertence a um conjunto, Ex: {1,2,3,4} ∉ A


Usado para comparações entre conjuntos:

Está contido: ⊂

Quando um conjunto está contido em outro, Ex: A = {1,2,3,4}, B = {1,2,3,4,5} | A ⊂ B


Não está contido: ⊄

Quando um conjunto não está contido em outro, Ex: A = {1,2,3,4}, B = {a,b,c,d} | A ⊄ B
Nota: é importante lembrar que se um conjunto está contido em outro, o que contém nunca vai estar contido no mesmo.


Contém: ⊃

Usando o mesmo exemplo de antes, B ⊃ A


Não contém: ⊅

Usando de novo do mesmo exemplo, A ⊅ B


Operações entre conjuntos:

União / Reunião: ∪

É unir os componentes de dois conjuntos, Ex: A = {1,2,3,4}, B = {5,6,7,8} | A ∪ B = {1,2,3,4,5,6,7,8}


Intersecção: ∩

É o que os dois conjuntos tem em comum, Ex: A = {1,2,3,4} B = {1,2,3,4,5,6} | A ∩ B = {1,2,3,4}


Diferença: -

É o que o primeiro conjunto tem, menos a intersecção do segundo, então, (Usando o mesmo exemplo de conjuntos de antes): B - A = {5,6}
